## Cumulative Relative Frequency Table

Given the following data:

2,3,8,5,3,2,6,9,3,5,8,2,5,6,3,3,9,2,5,3,6,3,2,9,3

v=value, f=frequency, Rf=relative frequency, CRf=cumulative relative frequency

| v   | f   | Rf        | CRf |
| --- | --- | --------- | --- |
| 2   | 5   | 5/25=0.20 | .20    |
| 3   | 8   | 0.32      |.52     |
| 5   | 4   | 0.16      |.68     |
| 6   | 3   | 0.12      |.80     |
| 8   | 2   | 0.08      |.88     |
| 9   | 3   | 0.12      |1.00     |
|     | 25  |  1.00         |     |
